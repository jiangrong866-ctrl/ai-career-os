# Daily Loop

Run one compact loop every day.

## 1. Input

Collect or import:

- Up to 10-20 AI-related jobs from Chrome-readable pages or manual import.
- New HR or interview signals from Gmail only when explicitly requested.
- Existing notes, prompts, project materials, and prior reports from Drive.

## 2. Analysis

For each job candidate, extract:

- Job title
- Company or source
- Requirements
- Salary if visible
- Match score
- Skill gaps

For knowledge items, extract:

- Category: JD, Resume, Project, Script, or Prompt
- Up to 5 key points
- Use: employment, side_business, portfolio
- Privacy label

## 3. Decision

Choose exactly:

- Top 3 jobs or fewer
- 1 learning task
- 1 side-business content or service task
- 1 portfolio improvement

## 4. Execution

Produce:

- Job scan summary
- Skill learning micro-task
- Side-business output
- Portfolio update instruction or draft

Do not auto-apply to jobs, auto-send emails, or publish public pages without explicit confirmation.

## 5. Output

Write a daily report with this shape:

- New jobs
- Updated jobs
- S/A grade counts
- Top 3 jobs
- Top 3 high-frequency skills
- New capability gap
- One learning task
- One side-business output
- One portfolio suggestion
- Up to 3 action suggestions
